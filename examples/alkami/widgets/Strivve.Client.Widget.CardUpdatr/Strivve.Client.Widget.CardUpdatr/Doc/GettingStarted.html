Getting Started

Alkami Widget

Setup your widget

The widget template creates an MVC structured class library project that includes references to the required NuGet packages

First Steps

Some things will still need to be completed:

  1. Update the Alkami.Client and Alkami.Common packages. Look at "Installed Packages" under the NuGet Package Manager and update these versions to match the version of ORB you are targeting.
  2. Edit and run the "install_widget.sql" script in the "Tools" directory, this will the widget by inserting a few rows into a few tables of the DeveloperDynamic database.

Updating Packages

Update the Alkami.Client and Alkami.Common NuGet packages as your staging environment is updated. Most times the libraries will be available before they are released to stage. Best practice is to stay in sync with your higher environments until a new Client or Common release is available for testing.
